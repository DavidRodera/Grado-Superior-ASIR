Una computadora, computador u ordenador es una máquina programable capaz de ejecutar una serie de comandos lógicos y aritméticos con gran precisión. Su principal función es procesar los datos de entrada, transformándolos en información útil que luego puede ser almacenada, visualizada o transmitida a otros dispositivos. Este proceso se realiza de forma automática, siguiendo instrucciones previamente definidas por el usuario o por el sistema. Una computadora está compuesta por numerosos y complejos circuitos integrados, así como diversos componentes electrónicos y periféricos que le permiten interactuar con el entorno. Además, cuenta con elementos de apoyo, extensión y accesorios que amplían sus capacidades operativas. Todas estas funciones se llevan a cabo bajo el control de programas informáticos conocidos como software, los cuales definen qué tareas se deben realizar y cómo ejecutarlas. Gracias a esta combinación de hardware y software, las computadoras pueden desarrollar múltiples tareas con gran velocidad y eficiencia.

Componentes y Funcionamiento Básico de una Computadora

La computadora está compuesta por dos partes esenciales que trabajan en conjunto para su funcionamiento. La primera es el hardware, que representa la estructura física del sistema: incluye todos los componentes tangibles como los circuitos electrónicos, cables, el gabinete, el teclado, el ratón, el monitor, y otros dispositivos periféricos. Estos elementos permiten la ejecución de operaciones físicas y la interacción del usuario con la máquina. La segunda parte es el software, que constituye la parte intangible del sistema. Este abarca los programas, aplicaciones, datos, documentación y sistemas operativos que le indican al hardware qué tareas debe realizar. Sin el hardware, el software no podría ejecutarse, y sin el software, el hardware no tendría instrucciones que seguir. Ambos componentes son fundamentales e inseparables para el funcionamiento eficiente de cualquier computadora moderna.

Desde el punto de vista funcional es una máquina que posee, al menos, una unidad central de procesamiento (CPU), una unidad de memoria y otra de entrada/salida (periférico). Los periféricos de entrada permiten el ingreso de datos, la CPU se encarga de su procesamiento (operaciones aritmético-lógicas) y los dispositivos de salida los comunican a los medios externos. Es así, que la computadora recibe datos, los procesa y emite la información resultante, la que luego puede ser interpretada, almacenada, transmitida a otra máquina o dispositivo o sencillamente impresa; todo ello a criterio de un operador o usuario y bajo el control de un programa de computación.

Datos
Es cualquier secuencia de uno o más símbolos a los que se les da significado mediante actos específicos de interpretación.
Hardware
En informática se refiere a las partes físicas, tangibles, de un sistema informático, sus componentes eléctricos, electrónicos y electromecánicos, los periféricos de todo tipo, y cualquier otro elemento físico involucrado.
Software
Sistema formal de un sistema informático, que comprende el conjunto de los componentes lógicos necesarios que hace posible la realización de tareas específicas.
