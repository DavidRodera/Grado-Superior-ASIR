# ASIR - Apuntes y Trabajos

Repositorio con apuntes, prácticas y trabajos del ciclo formativo de Administración de Sistemas Informáticos en Red (ASIR).

## Contenido

Este repositorio contiene:

- Apuntes teóricos de las asignaturas
- Prácticas de laboratorio
- Trabajos y proyectos

## Objetivo

Este repositorio sirve como recopilación de material de estudio del ciclo ASIR y como referencia para otros usuarios.
